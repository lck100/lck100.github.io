【异常-Git-3】fatal_ remote origin already exists.

错误

原因

想要添加远程仓库，但是远程仓库已经存在，且不是自己想要添加的远程仓库。

解决

通过git remote -v命令查看远程仓库信息

解决就是删除相关联的远程仓库通过git remote rm origin命令，再添加远程仓库即可。

【异常-Hexo-2】YYAMLException_incomplete explicit mapping pair; a key node is missed; or followed by a non-tabulated empty line at line 1

Posted on 2020-10-17

错误1

原因1

是title的冒号后面不应该有空格。

解决1

不留空格。

（注：这个解决方法虽然能够解决控制台报错问题，但是浏览器浏览博客标题无法显示正常，所以说还是需要空格的。即title和date在冒号后都需要空格将后面的内容隔开。）

错误2

原因2

注意title中后面又出现了英文冒号，产生了错误。

解决2

将英文冒号换成中文，或者其他的符号。

（注：虽然能解决控制台报错，但不能解决serve浏览器无法显示的问题，所以改成其他合法的符号，如下划线）。

有时候设置个空格也可能报错，所以视情况而定。反正出现这种类型的错误，就是title和date后面的内容格式不标准。

【异常-SpringBoot-2】org.apache.catalina.LifecycleException_ Failed to start component [Connector[HTTP/1.1-8081]]

Posted on 2020-10-17

错误

原因

项目端口不是8081端口，并且8081端口被占用。

解决

将端口由8080改为8081

同时终止8081端口的占用进程。

使用netstat -aon|findstr 8081命令查看占用8081端口的进程。

使用taskkill /pid 41712 /f命令终止PID为41712的进程，这里的41712是所占用进程的PID，是通过上面的命令查询得来的。

最终项目运行成功。
