【异常-Git-1】Permanently added the RSA host key for IP address 'xxx' to the list of known hosts.

Posted on 2020-10-16

错误

从git上克隆项目报错：

原因

SSH key出问题了。

解决

打开git bash，输入ssh -T git@github.com命令，查看能够与GitHub连接上。

发现报错提示：git@github.com: Permission denied (publickey).

输入**ls ~/.ssh/**命令，发现公钥文件已经存在

如果不存在则重新生成新的SSH key。

具体如何生成这里不详述，可以参考博客。

SSH key文件一般在C:\Users\Administrator.ssh目录下，如果没有可以自行在电脑搜索。

复制id_rsa.pub文件内的内容，到GitHub

点击New SSH key按钮，新建SSH key。

然后输入密码进行验证

然后生成成功

再次运行ssh -T git@github.com命令，输出就不同了。

就能克隆GitHub上的项目了。
